Class Deque<T>

A double ended queue using a double linked list, where .popLeft() and .appendLeft() will be a time complexity of O(1), as opposed to O(n) for .shift() and .unshift().

Example (es imports)

 
import { Deque } from 'spectrum-kit';

const deque = new Deque<number>([1, 2, 3]);

Example (commonjs)

 
var Deque = require('spectrum-kit').Deque;

var deque = new Deque<number>([1, 2, 3]);

Type parameters

  • T

    The type of the value stored in the deque
